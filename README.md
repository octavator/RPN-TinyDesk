# RPN-TinyDesk
Reversed Polish Notation Calculator

---           Start Program       ---

For python 3 users
``` py main.py ```  

For python 2 users
``` py main.py -p2 ```


Manages up to 4 different kind of mathematical operations and 3 specific text commands:
-> +
-> -
-> *
-> / 
-> help
-> clear
-> quit
